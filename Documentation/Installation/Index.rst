Installation
============

..  _requirements:

Requirements
============

-   PHP 8.2 - 8.5
-   TYPO3 13.4 LTS - 14.0

..  _steps:

Installation
============

Require the extension via Composer (recommended):

..  code-block:: bash

    composer require eliashaeussler/typo3-warming

Or download it from the
`TYPO3 extension repository <https://extensions.typo3.org/extension/warming>`__.
